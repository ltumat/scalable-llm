# Qwen3-4B-Instruct-FineTome

## How to run locally

```bash
uv sync               # install dependencies
uv run python -m inference  # launch Gradio app locally
```

## Model

Uses `alvaro-mazcu/Qwen3-4B-Instruct-FineTome` (fine-tuned Qwen3-4B-Instruct-2507). Chat UI includes temperature and max-token controls.
